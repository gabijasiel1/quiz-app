# Quiz App

Simple React Native quiz app for the assignment.

## Includes

- React Native Navigation
- `Question` component exported from `App.js`
- `Summary` component exported from `App.js`
- True/False, Multiple Choice, and Multiple Answer questions
- Total score on the summary screen

## Run

```bash
npm install
npm start
```

## GitHub Pages

```bash
npm run prepare-pages
```

This creates a `docs` folder ready for GitHub Pages.
